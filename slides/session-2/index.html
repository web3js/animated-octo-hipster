Public, Private and the rest of the story

When writing programs, we want to divide our code sensible modules. We call these "classes". This is how a class looks like in C++ 


// A class in C++
class person {

public:
  // public variables and methods, accessible from outside

private :
  // private variables and methods, accessible only from inside the class 

};

Well guess what?

That's right. In JS we don't have that.

But why do we care?

We care, because when our code grows in complexity and size, if we are all over the place, there is no way we can maintain and read our code. Let alone improve it and add to it.

So we will talk about classes, public and private members.

var calculate = function(num1, num2){

Immediately Executed Function Expressions

Her friends call her IIFE. She defines AND executes functions right then and there. 


var myObj = (function(var1, var2, var3, var4) {
  return {
    foo : var1,
    bar : var2,
    baz : var3,
    boing : var4
 }	
})('beer', 'coffee', 'juice','water');
console.log(myObj);
Read more about IIFE

Why do I care?

Well! If you wanna be an Object Oriented programmer, that's the way to go in JS. IIFE's are convenienct ways for us to create privacy in JS. Remember we talked about how Functions are the only thing in JS that create scope? The scope created by IIFE's can create Scope private and public members for us.

Let's make things private.

Try something like this:

Privileged

Privileged members bridge private and public. Using them, we can control what goes out and what stays in.
Have you ever heard of "getters" and "setters" ?
Those are privileged methods too. Watch: 


var myObj = (function() {
  
  var firstName = 'Mani',
    lastName = 'Nilchiani',
    age = '27',
    getFirstName = function(){
      return firstName;
    }

  return {
    getFirstName : getFirstName
  }

})();
console.log(myObj);
Read more Getters and Setters in JS

Privileged contd.

Now let's go ahead and create some setters too: With a little help from our friend Closure. 


var myObj = (function() {
  
  var firstName = 'Mani',
    lastName = 'Nilchiani',
    age = '27',
    getFirstName = function(){
      return firstName;
    },
    setFirstName = function(newName) {
      firstName = newName;  
    };

  return {
    getFirstName : getFirstName,
    setFirstName : setFirstName
  }

})();
console.log(myObj);
